# Mongo Bookstore

A simple MongoDB project simulating an online bookstore.  
Supports viewing books, adding new books, and placing orders.  
Includes an interactive Mongo shell menu for hands-on practice with CRUD operations.

## Features
- View all books in the store
- Add new books
- Place orders for customers
- Automatically updates book stock

1. Install MongoDB or use MongoDB Atlas.
